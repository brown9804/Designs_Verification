# Qflow Analysis

Aspects analyzed:

*. Frequency.

*. Area.

*. Number and type of gates.

*. Path delays.

*. Place and route.

## Dependencies

Make sure qflow is install:

~~~~
$ sudo apt-get update
$ sudo apt-get install qflow
~~~~

* yosys:

Verilog parser/synthesis (yosys)
* graywolf:

Cell and pin placement

* qrouter 1.3:

Detail router

* magic 8.1:

Layout viewer
